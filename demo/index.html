RF-DenseNet Architecture

Architecture Overview

1
Input Layer: 64×64×3 RF Spectrogram
2
Initial Conv: 3×3, 16 filters, BatchNorm, ReLU
3
Dense Blocks: 3 blocks, 3 layers each, k=8
4
Transitions: 1×1 conv, θ=0.5 compression, 2×2 pool
5
Classification: Global Avg Pool, Dropout, Softmax
